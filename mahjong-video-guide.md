## 🎯 AI 客服使用指南

### 什麼時候提供影片連結？

#### ✅ 適合提供影片
1. **基本操作問題**（開機、續時、門把）
2. **設定調整**（有花/無花、風乾）
3. **簡單故障**（骰子、卡牌）
4. **客人說「我想自己試試看」**

#### ⚠️ 謹慎提供影片
1. **進階維護**（小蝸牛組裝、圓盤拆卸）
2. **異物排除**（可能造成損壞）
3. **客人很急或很生氣**（直接安排技術人員）

#### ❌ 不提供影片，直接轉真人
1. **冒煙、異味、漏電**（緊急狀況）
2. **客人嘗試過影片方法但無效**
3. **問題描述不清楚**（無法判斷適合哪個影片）

## 🔄 完整處理流程

```
客人回報麻將桌問題
    ↓
【判斷 1】緊急狀況？
├─ 是 → 不提供影片，立即轉真人
└─ 否 ↓

【判斷 2】問題類型？
├─ 基本操作 → 提供影片 + 文字說明
├─ 設定調整 → 提供影片 + 提醒重開機
├─ 簡單故障 → 提供影片 + 後續支援
└─ 進階維護 → 建議轉真人（可附影片參考）
    ↓

【判斷 3】客人看完影片後反饋？
├─ 解決了 → 「太好了！有問題隨時找我」
├─ 還是不行 → 「我幫您請技術人員」
└─ 看不懂 → 「我用文字再說明一次」或轉真人
```
