# 商家 Sheets 自動建立指南

**⚡ 一鍵建立完整的商家設定 Sheets**

每個商家一個獨立的 Google Sheets，包含完整的規則設定、店家資訊。

---

## 🚀 使用方式（3分鐘）

### 步驟 1：新建 Google Sheets（30秒）
1. 前往 https://sheets.google.com
2. 點選「空白試算表」

### 步驟 2：開啟 Apps Script（30秒）
1. 在 Sheets 點選上方選單「擴充功能」→「Apps Script」
2. 會開啟新視窗

### 步驟 3：貼上腳本（1分鐘）
1. 刪除預設的 `function myFunction() { ... }`
2. 從 [merchant-auto-setup.gs](./merchant-auto-setup.gs) 複製完整代碼
3. 貼上到編輯器
4. 點選「儲存」（💾）

### 步驟 4：執行腳本（1分鐘）
1. 點選編輯器上方的函數選單
2. 選擇 `createMerchantSheets`
3. 點選「執行」（▶️）
4. 第一次執行會要求授權（點選「允許」）
5. 等待執行完成（約10-15秒）
6. 會彈出提示「✅ 建立完成！」

### 步驟 5：取得 Sheets ID（30秒）
1. 點選函數選單，選擇 `getSheetId`
2. 點選「執行」
3. 會彈出視窗顯示 Sheets ID
4. **複製這個 ID**

## 🔄 規則處理邏輯

### 三層規則架構

```
【第一層】總部核心規則（優先級 9-10）
  從總部 Sheets 讀取
  無條件執行
     ↓
【第二層】商家自訂規則（優先級 7-8）
  從商家 Sheets「商家規則」讀取
  檢查觸發條件 → 符合則執行
     ↓
【第三層】店家資訊查詢
  從商家 Sheets「店家資訊」讀取
  回答價格、Wifi、地址等
     ↓
【最終】AI 生成 + 禁止詞檢查
```

## ✏️ 建立後需要修改的地方

### 必改項目 ⚠️

1. **商家資訊**分頁：
   - 商家ID（改成你的 ID）
   - 商家名稱（改成實際名稱）
   - 聯絡電話（改成實際電話）
   - **總部Sheets ID**（填入總部規則庫 ID）

2. **店家清單**分頁：
   - 刪除範例店家
   - 新增你的實際店家

3. **店家資訊**分頁：
   - 刪除範例店家資訊
   - 填入你的實際店家資訊（價格、地址、Wifi等）

### 可改項目

1. **AI設定**：
   - AI個性（5種選擇）
   - 回話字數限制
   - 首購優惠、整潔換券

2. **商家規則**：
   - 新增更多商家規則
   - 修改觸發條件
   - 調整優先級

3. **退單政策**：
   - 提前取消時限
   - 扣款比例
   - 特殊情況處理
