# 總部 Google Sheets 架構設計

**版本**: v1.0  
**用途**: AI客服系統總部規則庫

---

## 📋 Sheets 結構總覽

建立 7 個分頁：

| 分頁名稱 | 用途 | 欄位數 |
|---------|------|--------|
| 核心規則 | 統一管理核心規則 | 4 |
| 標準SOP | 通用故障排除流程 | 5 |
| 商家清單 | 管理所有商家資訊 | 8 |
| 白名單關鍵字 | 驗證客人身分 | 3 |
| 禁止詞庫 | AI絕對不可說的詞 | 3 |
| 系統設定 | 全域參數 | 2 |
| 更新紀錄 | 追蹤修改歷史 | 5 |

---

## 📄 分頁 1：核心規則

### 欄位結構
```
A1: 規則ID
B1: 規則名稱
C1: 規則內容
D1: 優先級
```

### 範例資料
```
A2: CORE_001
B2: 無人店身分設定
C2: 你是ONE桌遊的AI客服，這是無人店，沒有現場人員。嚴禁提及「服務員」「店員」「工作人員」等詞彙。
D2: 10
```

## 🚀 快速建立步驟

### 1. 新建 Google Sheets
1. 前往 https://sheets.google.com
2. 點選「空白試算表」
3. 重新命名為「ONE桌遊AI客服-總部規則庫」

### 2. 建立 7 個分頁
點擊左下角「+」新增工作表，依序建立：
- 核心規則
- 標準SOP
- 商家清單
- 白名單關鍵字
- 禁止詞庫
- 系統設定
- 更新紀錄

### 3. 複製欄位標題
在每個分頁的第一行填入對應的欄位標題（參考上方結構）

### 4. 填入測試資料
至少在「商家清單」填入一筆測試資料，確保系統能正常運作

### 5. 複製 Sheets ID
從網址列複製 Sheets ID：
```
https://docs.google.com/spreadsheets/d/【這一段就是ID】/edit
```
