## 📋 常見問題分類

### 🔴 緊急問題（立即通知真人）
- 冒煙、異味、異常高溫
- 漏電、電擊感
- 嚴重卡牌無法取出

### 🟡 中度問題（提供 SOP，無效則轉真人）
- 張數不對
- 卡牌、洗牌很久
- 骰子不會跳
- 升降台故障

### 🟢 輕微問題（AI 可自行處理）
- 混牌（異色牌混在一起）
- 操作不熟悉
- 檔位設定錯誤

## 🛠️ 問題 1：剛開機張數不對

### 常見現象
- 一開局就少牌或多牌
- 某家特別多或特別少
- 整副牌數量錯誤

### 可能原因
1. **檔位開關設定錯誤**（最常見）
2. **異色牌混入**
3. **牌卡在機器內部**
4. **感應器故障**

### 排除步驟

#### 步驟 1：檢查檔位開關（90%問題在這）
```
1. 找到麻將桌側面的「檔位開關」
2. 確認設定在「自動」檔（通常是中間位置）
3. 常見錯誤：
   - 設定在「手動」→ 改回「自動」
   - 設定在「測試」→ 改回「自動」
   - 開關在兩檔之間 → 推到定位
```

#### 步驟 2：檢查是否有混牌
```
1. 觀察四副牌的顏色
2. 常見：紅色牌混入綠色組（或其他顏色互混）
3. 如果有混牌：
   - 暫停使用該組牌
   - 手動挑出異色牌
   - 歸回正確牌組
   - 重新開機測試
```

#### 步驟 3：重新啟動
```
1. 關機（關閉電源開關）
2. 等待 30 秒
3. 重新開機
4. 按「開始」鍵測試
```

#### 步驟 4：檢查牌數
```
1. 如果仍然不對，手動清點牌數
2. 標準配置：
   - 一般麻將：144 張（每色 36 張）
   - 十六張麻將：148 張
   - 十三張麻將：136 張
3. 如果少牌：檢查桌面下方、升降台
4. 如果多牌：檢查是否有多副牌混在一起
```

### 無效時的處理
```
若以上步驟都無效：
1. 拍照記錄（牌數、檔位設定）
2. 記錄桌號
3. 聯絡客服：「麻將桌故障，已排除但仍張數不對」
4. 客服會安排技術人員處理
```

### AI 客服回覆範本
```
收到！麻將桌張數不對，我來幫您排除：

【步驟 1】檢查檔位開關
請找桌子側面的檔位開關，確認是否在「自動」檔。
常見問題：開關在「手動」或「測試」檔。

【步驟 2】檢查混牌
觀察四副牌顏色，如果有異色牌（例如紅牌混入綠牌），請手動挑出後重新開機。

【步驟 3】重新啟動
關機 → 等 30 秒 → 重新開機 → 測試

如果以上步驟都試過還是不行，請拍照桌號並告訴我，我會立即請技術人員協助！
```

## 🛠️ 問題 2：卡牌 / 洗牌很久沒出來

### 常見現象
- 按開始後，聽到馬達聲但牌一直不出來
- 升降台卡在中間
- 洗牌聲音很久（超過 2 分鐘）

### 可能原因
1. **牌卡在輸送帶**（最常見）
2. **升降台感應器故障**
3. **馬達過熱保護**
4. **牌太舊變形**

### 排除步驟

#### 步驟 1：長按骰子鍵重置（80%問題解決）
```
【關鍵步驟】
1. 找到桌面上的「骰子鍵」（通常左上和右下各一個）
2. 同時長按「對角的兩個骰子鍵」
   - 左上 + 右下，或
   - 右上 + 左下
3. 長按約 5 秒，聽到「嗶」一聲後放開
4. 等待機器自動重置（約 10-15 秒）
5. 重新按「開始」鍵

【為什麼有效】
這個操作會清空機器內部的牌，重新初始化系統。
```

#### 步驟 2：檢查升降台
```
1. 關機
2. 觀察升降台是否在最下方
3. 如果卡在中間：
   - 輕輕往下壓（不要太用力）
   - 確認沒有牌卡住
4. 重新開機
```

#### 步驟 3：檢查異常聲音
```
正常聲音：
- 馬達運轉聲（嗡嗡聲）
- 牌洗動聲（嘩啦聲）

異常聲音：
- 咔咔聲 → 可能有牌卡住
- 尖銳摩擦聲 → 可能有異物
- 完全沒聲音 → 電源或馬達問題
```

#### 步驟 4：等待冷卻
```
如果剛連續使用很久：
1. 關機休息 10-15 分鐘
2. 馬達可能過熱啟動保護
3. 冷卻後重新開機
```

### 無效時的處理
```
若長按骰子鍵無效：
1. 不要強制開啟桌面（避免損壞）
2. 記錄桌號、使用時長
3. 拍照升降台狀態
4. 立即聯絡客服
```

### AI 客服回覆範本
```
了解！麻將桌卡牌了，我教您一個很有效的方法：

【長按骰子鍵重置】（80%可以解決）
1. 找到桌面上的骰子鍵（左上、右上、左下、右下各一個）
2. 同時長按「對角的兩個骰子鍵」約 5 秒
   例如：左上 + 右下 一起按
3. 聽到「嗶」一聲後放開
4. 等待機器重置（約 10-15 秒）
5. 重新按「開始」鍵

如果還是不行，可能需要休息 10 分鐘（馬達過熱保護），或是我幫您請技術人員過去檢查。

試試看，有問題隨時跟我說！
```

## 🛠️ 問題 3：混牌（異色牌混在一起）

### 常見現象
- 發牌時發現異色牌
- 例如：紅色牌混入綠色組
- 四副牌顏色不一致

### 可能原因
1. **客人不小心混牌**（最常見）
2. **前一組客人留下的**
3. **機器洗牌時誤送**

### 排除步驟

#### 步驟 1：立即暫停
```
1. 不要繼續使用該組牌
2. 避免混牌更嚴重
```

#### 步驟 2：手動挑出異色牌
```
1. 仔細檢查這副牌
2. 找出所有異色牌
3. 常見混法：
   - 紅牌 → 綠牌
   - 綠牌 → 藍牌
   - 藍牌 → 黃牌
```

#### 步驟 3：歸回正確牌組
```
1. 將異色牌放回對應顏色的牌組
2. 確認四組顏色都正確
3. 重新數一下張數（每組應該相同）
```

#### 步驟 4：重新開機
```
1. 關機
2. 等待 30 秒
3. 重新開機
4. 測試新一局
```

### 預防措施
```
提醒客人：
- 遊戲結束後不要隨意移動牌
- 如果不小心掉牌，請放回原位
- 不要將不同桌的牌混在一起
```

### AI 客服回覆範本
```
收到！有異色牌混在一起了。

【處理方式】
1. 暫停使用這副牌
2. 手動挑出異色牌（例如紅牌混入綠牌）
3. 將異色牌放回正確的牌組
4. 重新開機測試

這通常是前一組客人不小心混到的，挑出來就可以了。

如果挑完還是有問題，或是不確定怎麼分辨顏色，拍照給我看，我幫您確認！
```

## 🛠️ 問題 4：骰子不會跳

### 常見現象
- 按開始後骰子不動
- 骰子只亮燈不跳
- 骰子跳一下就停

### 可能原因
1. **電池沒電**（最常見）
2. **骰子開關沒開**
3. **接觸不良**
4. **骰子故障**

### 排除步驟

#### 步驟 1：檢查骰子開關
```
1. 骰子底部通常有小開關
2. 確認開關在「ON」位置
3. 如果在「OFF」→ 撥到「ON」
```

#### 步驟 2：更換電池
```
1. 打開骰子底部電池蓋
2. 常見電池型號：
   - AAA（4號電池）
   - AA（3號電池）
   - CR2032（鈕扣電池）
3. 更換新電池
4. 注意正負極方向
```

#### 步驟 3：清潔接觸點
```
1. 取出電池
2. 用乾布擦拭電池接觸點
3. 去除氧化或灰塵
4. 重新裝上電池
```

#### 步驟 4：手動搖骰子
```
如果骰子還是不跳：
1. 暫時用手動方式決定莊家
2. 或用手機 APP 骰子（替代方案）
3. 遊戲可以繼續進行
```

### 無效時的處理
```
若以上都無效：
1. 記錄骰子狀態（完全不亮/只亮不跳）
2. 記錄桌號
3. 聯絡客服更換骰子
4. 通常 30 分鐘內可處理
```

### AI 客服回覆範本
```
骰子不跳了是嗎？我幫您檢查：

【最常見原因：電池沒電】
1. 打開骰子底部電池蓋
2. 更換新電池（通常是 4 號電池）
3. 注意正負極方向
4. 蓋回電池蓋

【其他可能】
- 骰子底部開關沒開 → 撥到 ON
- 電池接觸不良 → 擦拭接觸點

如果沒有備用電池，可以暫時用手機 APP 骰子，或我幫您請客服送電池過去（約 20 分鐘）。

需要我協助嗎？
```

## 🛠️ 問題 5：升降台不會升/降

### 常見現象
- 按開始後升降台沒反應
- 升降台卡在中間
- 升降台上升後不下降

### 可能原因
1. **牌沒清空**
2. **感應器被遮擋**
3. **馬達故障**
4. **機械卡住**

### 排除步驟

#### 步驟 1：清空升降台
```
1. 確認升降台上沒有牌
2. 常見問題：有幾張牌掉在升降台
3. 手動移除所有牌
4. 重新按開始
```

#### 步驟 2：檢查感應器
```
1. 升降台周圍有小孔（感應器）
2. 確認沒有被牌、垃圾遮擋
3. 用手電筒照一下，清除灰塵
```

#### 步驟 3：手動輔助
```
1. 關機
2. 輕輕往下壓升降台（不要太用力）
3. 確認沒有卡住
4. 重新開機
```

#### 步驟 4：長按重置
```
1. 同時長按對角骰子鍵 5 秒
2. 系統重置
3. 重新測試
```

### 無效時的處理
```
若升降台完全不動：
1. 不要強制操作（避免損壞）
2. 記錄桌號
3. 立即聯絡客服
4. 技術人員會帶工具處理
```

### AI 客服回覆範本
```
升降台卡住了，我幫您排除：

【步驟 1】清空升降台
確認升降台上沒有任何牌（有時會有幾張掉下去）

【步驟 2】長按骰子鍵重置
同時長按對角的兩個骰子鍵 5 秒 → 聽到「嗶」一聲 → 放開

【步驟 3】手動輔助
如果還是不行，關機後輕輕往下壓升降台，確認沒卡住，再重新開機。

如果升降台完全不動，請不要強制操作，我立即幫您請技術人員處理（約 20 分鐘內到）。

先試試看，有問題隨時跟我說！
```

## 🚨 緊急狀況處理

### 🔴 冒煙 / 異味 / 高溫

#### 立即動作
```
1. 【立即關機】拔掉電源插頭
2. 【疏散人員】遠離麻將桌
3. 【通報客服】立即 LINE 通知
4. 【不要碰觸】等待技術人員
```

#### AI 客服回覆
```
⚠️ 請立即關機並拔掉電源插頭！

我已緊急通知技術人員，請您：
1. 遠離麻將桌
2. 保持 LINE 暢通
3. 客服會立即致電給您（3 分鐘內）

您的安全最重要，千萬不要嘗試自行修理！
```

---

### 🔴 漏電 / 電擊感

#### 立即動作
```
1. 【停止使用】不要再碰桌子
2. 【關閉電源】拔掉插頭（用絕緣物品）
3. 【立即通報】LINE 通知客服
4. 【貼告示】「故障維修中」
```

#### AI 客服回覆
```
⚠️ 這很危險！請立即停止使用！

我已緊急通知技術人員和主管。
請您：
1. 不要再碰觸麻將桌
2. 拔掉電源插頭（用絕緣物品，如木棍）
3. 保持 LINE 暢通
4. 緊急聯絡人會立即致電（3 分鐘內）

非常抱歉造成您的困擾，我們會立即處理！
```

## 📊 AI 客服判斷流程圖

```
客人回報麻將桌問題
    ↓
【檢查 1】是否為緊急狀況？
├─ 冒煙/異味/高溫 → 立即推播 + 緊急回覆
├─ 漏電/電擊感 → 立即推播 + 緊急回覆
└─ 否 ↓

【檢查 2】問題類型？
├─ 張數不對 → 提供檔位檢查 SOP
├─ 卡牌/洗很久 → 提供長按骰子鍵 SOP
├─ 混牌 → 提供手動挑牌 SOP
├─ 骰子不跳 → 提供換電池 SOP
└─ 升降台故障 → 提供清空+重置 SOP
    ↓

【檢查 3】客人反饋「試過了還是不行」？
├─ 是 → 推播通知真人客服 + 「技術人員協助」回覆
└─ 否 → 繼續提供 SOP + 「隨時跟我說」

【檢查 4】客人再次回報「還是不行」？
└─ 立即推播真人客服 + 「馬上安排技術人員」
```

## 🎯 AI 客服回覆原則

### ✅ 要做的
1. **先安撫情緒**：「收到！我幫您排除」
2. **分步驟說明**：用【步驟 1】【步驟 2】格式
3. **提供原因**：「通常是...造成的」
4. **給予信心**：「80%可以解決」
5. **隨時支援**：「有問題隨時跟我說」
6. **及時升級**：試過 2 次不行就轉真人

### ❌ 不要做的
1. ~~「這很正常」~~（客人會生氣）
2. ~~「自己修修看」~~（太不負責）
3. ~~「應該是...吧」~~（不確定的話）
4. ~~讓客人等太久~~（超過 2 次就轉真人）
5. ~~使用技術術語~~（用白話文）
6. ~~推卸責任~~（「可能是你操作不當」）

## 📞 客服升級時機

### 立即升級（不提供 SOP）
- 冒煙、異味、高溫
- 漏電、電擊感
- 客人受傷

### 提供 SOP 後升級
- 客人嘗試 2 次仍無效
- 客人表達「很急」「很生氣」
- 問題描述不清楚（無法判斷）

### 可延後升級
- 混牌（必定可自行解決）
- 骰子不跳（可用替代方案）
- 操作問題（教學即可）
