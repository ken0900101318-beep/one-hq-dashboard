# ONE桌遊 AI客服系統 - 總部戰情室 v1.1

**🎮 總部戰情室管理介面**

監控 90 家店的 AI 客服系統，管理核心規則、標準SOP、白名單與禁止詞。

---

## 🚀 快速開始

### 1️⃣ 開啟戰情室

👉 **直接開啟**: https://ken0900101318-beep.github.io/one-hq-dashboard/

### 2️⃣ 建立總部 Google Sheets

參考 [總部 Sheets 架構設計](./sheets-setup.md) 建立總部規則庫。

需要 7 個分頁：
- 核心規則
- 標準SOP
- 商家清單
- 白名單關鍵字
- 禁止詞庫
- 系統設定
- 更新紀錄

### 3️⃣ 部署 Apps Script 後端

1. 在總部 Sheets 開啟「擴充功能」→「Apps Script」
2. 複製 [後端代碼](./backend-code.md)
3. 填入 `HQ_SHEET_ID`
4. 部署為 Web App
5. 複製 Web App URL

### 4️⃣ 連接戰情室

1. 開啟戰情室網址
2. 填入 Web App URL
3. 點選「🔗 連接後端」
4. 完成！

## 🐛 疑難排解

### 連接失敗
1. 確認 Web App URL 正確（以 `.../exec` 結尾）
2. 確認 Apps Script 已部署
3. 確認 `HQ_SHEET_ID` 已填入

### 規則顯示為空
1. 確認總部 Sheets 已填入資料
2. 確認分頁名稱正確
3. 執行 Apps Script 測試函數

### 權限錯誤
1. 在 Apps Script 執行測試函數
2. 重新授權
